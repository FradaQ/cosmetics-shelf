# 美妆库存 iPhone App

这是一个 SwiftUI + SwiftData 的 iOS 17 原型，用来管理护肤品、彩妆和香水库存。

## 已包含功能

- 按护肤、彩妆、香水、身体/头发、其他归档
- 记录品牌、产品名、批号、购买日期、生产日期、开封日期和备注
- 支持未开封保质期、开封后 PAO 月数、手动到期日
- 自动取最早建议到期日
- 到期前 6 个月进入提醒列表并安排本地通知
- 批号查询页可组织搜索词并打开网页查询
- 添加产品时可按名称查找公开产品数据库候选，并自动回填名称、品牌、图片链接和来源页
- 批号录入处预留自动解析入口；没有可靠规则时会提示手动输入生产日期或到期日

## 打开方式

用 Xcode 打开 `CosmeticsShelf.xcodeproj`，选择 iPhone 模拟器运行。

当前机器只启用了 Command Line Tools，无法在这里完成 `xcodebuild` 编译验证；工程文件格式已经通过 `plutil` 校验。
